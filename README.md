## 진행방식

- 온라인 - 행아웃으로 화면, 목소리 공유
- 요일 - 토/일 중 하루
- 시간 - 오전 11시부터 2시간 정도
- 풀고 싶은 사이트/문제 있으면 추천 부탁

## 규칙

- 1문제 풀이 시간 : 20분~30분
- 기본 20분으로 정하고, 모두가 못 풀었을 때는 최대 30분까지 같이 풀기
- 같은 문제를 풀고 제한시간이 지나면 한 사람씩 화면 공유해 풀이 방식 공유하기
- 못 풀었더라도 접근방식 공유하기
- 풀지 못한 문제가 있다면 다음 주까지 풀어오기
- 진행 방식은 상황에 맞춰 변경 가능
